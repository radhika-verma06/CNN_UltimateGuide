# 🧠 CNN Ultimate Guide  
### Interactive Deep Learning Visualisation Platform

CNN Ultimate Guide is an interactive educational platform designed to explain how Convolutional Neural Networks (CNNs) work through visual simulations and animated learning components.

Built entirely with **HTML, CSS, and JavaScript**, the project demonstrates core deep learning concepts such as convolution, pooling, training loops, dropout, bias-variance tradeoffs, and softmax prediction using interactive browser-based visualisations.

The platform is designed as a self-contained educational tool with zero external dependencies.

## ✨ Core Features

### 🔍 Interactive CNN Pipeline
Visualise the full CNN workflow:

- input image representation
- convolution filters
- feature map generation
- max pooling
- flattening
- dense layers
- softmax prediction

### 🎛️ Real-Time Convolution Visualisation
Includes:
- animated kernel scanning
- switchable filters
- live dot-product calculations
- feature map updates

### 📈 Training Loop Simulation
Demonstrates:
- forward propagation
- loss calculation
- backpropagation
- weight updates
- epoch progression

### 🎯 Bias vs Variance Demonstration
Interactive visual explanation of:
- underfitting
- good fit
- overfitting
- generalisation behaviour

### 🧩 Dropout Regularisation Simulator
Visualises:
- neuron dropout
- training vs inference behaviour
- adjustable dropout rates

### ❓ Interactive Quiz & FAQ
Includes:
- CNN concept quiz
- instant feedback
- expandable FAQ explanations
